# RadioTelescope

Open-source student-built radio telescope. Works (in theory).

## Information and build instructions

Our documentation is in the `docs/` directory. It uses the `mkdocs` static site generator with the [material theme](https://squidfunk.github.io/mkdocs-material/). To view it, install `mkdocs` with:

```bash
pip install mkdocs-material
```

Then, in the root folder, you can view the site with:

```bash
mkdocs serve
```

You should be able to see the instructions for building the telescope.

## Editing

The docs are just plaintext markdown files, so they can be edited in any code editor. We recommend using [Obsidian](https://obsidian.md/) - you can simply open the `docs/` folder as an Obsidian vault and edit as usual.
